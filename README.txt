CakeCost MVP — инструкция по запуску

1) Откройте командную строку и убедитесь, что у вас установлен Flutter и Android SDK.
2) В Android Studio выберите "Open Existing Project" и укажите папку C:\COST\cake_cost.
3) Дождитесь, пока будут подтянуты зависимости.
4) В панели Run выберите эмулятор или подключённый Android-телефон.
5) Нажмите ▶️ Run. Должен запуститься эмулятор и показать стартовый экран.

Если увидите экран с логотипом — значит всё работает!
